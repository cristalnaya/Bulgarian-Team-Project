The idea behind this project is each participant to create a webpage, using the knowledge from the Udacity Front End Web Developers challenge. Some of us had little or no previous experience with HTML, CSS and Javascript, so those are our very first steps in this immense and intriguing field. Why we started this project at all? Well like with human languages it is one thing to start learning, but a whole new level to dare using it. It is one thing to make the Udacity course projects by yourself at your ease, another to gather and work with different people on a common project, to share files via Github, to upload them online and all the little complications that come with it, that you don't think about while working alone on your local computer.
